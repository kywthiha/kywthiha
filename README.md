### 🔥 Recent Hackathon & Side Projects

**SupportPilot**  
Real-time multimodal AI customer support agent that can **see your screen**, **hear your voice**, and guide users step-by-step through complex SaaS platforms.  
Built with Gemini 2.5 Flash Live API, Google Agent Development Kit (ADK), Vertex AI RAG, FastAPI + WebSockets, React + TypeScript, and Docker.  
→ [View Repository](https://github.com/kywthiha/support-pilot)

**nextjs-agent-cli**  
AI-powered CLI tool that generates **production-ready fullstack Next.js applications** from natural language descriptions.  
It autonomously creates files, installs dependencies, sets up Prisma + PostgreSQL, verifies code, and fixes errors.  
Built for the Gemini 3 Hackathon 2026.  
→ [View Repository](https://github.com/kywthiha/nextjs-agent-cli)

**DuoMRReview**
AI-powered code review assistant developed during the **GitLab AI Hackathon**.  
Focused on improving code collaboration and review workflows using AI.  
→ [View Project](https://gitlab.com/gitlab-ai-hackathon/participants/35496680)
